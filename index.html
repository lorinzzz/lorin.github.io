Cursed Snake Game

You are now the food, and the snakes (plural!) are out there to eat you! A twist to the original snake game with many added features such as power-ups, boost ability, and game customization. One of the biggest takeway of this project is the collection of unique AIs powering each snake. Built using Python with the pygame library.

Proximity and Level Sensor Alarm

A proximity and level sensor with alarm triggering. Built with a Arrow DECA FPGA with a 4 digit 7seg display and a DC active buzzer.

Screenshot Organizer

Taking in-game screenshots using overlay applications such as MSI Afterburner or Nvidia Shadowplay usually results in a folder containing image files across all games. This script will read the image files and organize all the images into folders with proper naming.

  • Source Code

Discord Bot: Beautiful Sounds

Discord bot built for my friends and I. Essentially a sound board. Upon command the bot will join the current or directed voice channel and play the selected track before leaving the channel. Also has the option to repeat the track by repeatedly joining and leaving the channel.
